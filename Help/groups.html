Manage Groups Help Topics



This page is found under the Admin section.

To add a group click the Add link and complete the form.

Supply a unique Group ID, Name and set the status to Active or InActive.

Deleting a Group

Select the group, then press the delete button on the bottom of the form. Note: you should only delete groups after migrating any articles to other groups or removing all articles for the target group. You can bulk migrate articles from one group to another using the Manage Articles page.
