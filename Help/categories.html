Categories Help Topics


There are two Category groupings in the KB. Type and a Custom category that is used if the Administrator has enabled it via the Settings page.

To view/add or Modify a Category Type. Click on the Manage Categorization icon on the Admin page. You may add, modify or remove category Types.

Note: Do not remove category Types that are in use by active Articles, instead mark them as Inactive.
