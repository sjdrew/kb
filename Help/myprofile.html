Profile and Preferences Help Topics


The Profile page displays the current information about you, such as Email, Name, and phone number.

It also provides some Preferences that can be configured:

  • Default Search Group: Sets the default group to search on the Search and Browse pages. Set to All Groups if you wish to always search all groups (the default).
  • Default Search Mode: Choose either Strict or English Query. With Strict all words you enter are required in the article search, with English Query synonmys maybe used and noise words are removed.
  • Items per Page: Sets the number of items to be displayed per page.
  • Displau Article Previous: Affects the Search listing and whether it previous the first portion of the Article or just displays the article title.

Notifications:

You may enable or disable various notifications as listed on this page. For example if you do not want to receive Bulletin email message, you can uncheck the appropriate box.

If you are a Technical or Content reviewer, then you should check the appropriate boxes to ensure you receive email notification when articles are ready for review.
