Listing Command Bar Help Topics


This icon when clicked will reveal a Command Bar with operations that can be performed against the listing such as:

  • Select All - click this item to have all the list items selected (highlighted)
  • Select None - All items will be unselected.
  • Select between - All items between the first row you have highlighted and the next row you have highlighted will be selected. For example to select the first ten items, click the first then the tenth and then click "Select Between".
  • Display All - Unpaginate the current listing and display all items on one page.
  • Export List to Excel - Exports all items in the current list to an Excel file.
  • Modify Selected - Once you have selected the desired items, this command (which is enabled for certain listings that support this feature) enables you to modify fields on all the selected records in one operation. For more information on Modifying multiple records in one operation, please see Modifying Multiple Records.
  • Customize Columns - Allows you to customize the columns that are displayed for this particular listing. You may also define the order that the columns are displayed. Administrators may also define the system wide default listings used. This setting will be remembered each time you use KB.
