Modifying Multiple Records Help Topics


Using the Command Bar you can select multiple records and have the ability to change many items on these records in one operation.

For example, suppose a employee has left your company and was the person assigned to 50 Articles. Rather than updating these 50 Articles one at a time you can use this feature to modify them in one operation. The procedure for this is as follows:

  1. Find the Articles to modify, in this case use the Manage Articles dialog and select the assigned to droplist to the currently assigned person and press Find.
  2. The list of 50 items should be displayed. Click on the Command Bar icon and click Select All.
  3. Then click Modify Selected
  4. A Blank Article form is now displayed with the message that you are about to modify 50 Articles.
  5. Select the new person that should be assigned to these Articles using the Assign To droplist and press save.
  6. The 50 records are modified.

Note: If you have Pagination enabled when you chose the Select All it may of only selected the number of items displayed on Page 1 of your listing, if that is the case, then click Display All which will unpaginate the current listing and display all 50 items on one page. Then click Select All.
