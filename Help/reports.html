Reports Help Topics


Under the Admin page - Report icon, the following reports are available:

  • Article Read Report - Displays a list of Articles which have been read during a given time period. You can select by article, all articles and specify an account id that made the search.
  • Search History Report - Admin onluy report that displays searches and the resulting numer of matches. You can search the searches made by users by date range and account id and words uses within the search. For example you can find out how many searches for last month contained the word "Outlook".
  • Must Read Articles - Displays a list of Must Read Articles including Read/Unread status.
  • Usage Report - Displays a graphical representation of KB Activity per month
  • Activity Log Report - Admin report that displays all audited activity for a specific time or user. Events such as user login, article reads, articles modified, bulletins created etc.
  • MUI Report - Report of Multi User Incident Bulletins

Most reports can be exported using the Cmd Bar.
