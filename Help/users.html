Manage Users Help Topics



This page is found under the Admin section.

To manually add a user click the Add button and complete the form.

The UserID must be specified to match the Active Directory user name for the default domain or if the KB application is set to local account/password you must supply a userId and password combination that the user will be required to enter to access KB articles.

To add user to a group click the Add button beside the Group field and select the Group name and access permissions from the popup window.

Group access permissions:

  • Read - Access to read articles for this group that are specified as group only articles
  • Write - Write access to create articles for this group, but not permitted to publish.
  • Approval - Access to review and publish articles for this group.

Must Read checkbox:

  • If set, the user will have Must Read flagged articles displayed on their home page until they have read them.

Deleting a user

  • Click the delete button and once confirmed all information regarding the user account is removed.

User Activity

  • When displaying a user, number of links appear at the bottom of the page to provide quick reports regarding activity performed by the selected user.
