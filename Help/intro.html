The Service Desk Knowledge base provides a repository of Processes, How To's, Technical Articles and System Documentation.

Infromation is retrieved using an english query or key words. The search engine, by default returns the highed Ranked articles first.

You may change the sort order by clicking on any of the columns. Under the My Profile page you may set certain preferences, such as the number of items per page, the default query type and whether to display previews in the search results.

Query Types:

  • English - Noise words and synonyms are used when processing the query.
  • Strict - All the words or phrases entered must exist in the Article. You can wild card words for example Work* would match any word begining with work. (You cannot use the wildcard at the start of the word, for example *Station will not work).
  • Title - The search pattern is matched against the Title field only.
