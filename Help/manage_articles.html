Manage Articles Help Topics



This page is found under the Admin section.

It is used to search for articles regardless of the Article status. You can search for articles based various criteria, such as contacts, modified date, author etc.

The listed returned from this query can be used to perform bulk modifications to the the result set. For example you can query all Articles for a certain contact and then change the contact for all matched articles in one operation. See the help on the command bar for more information.
